para ejecutar el programa se debe usar el comando:
node principal.js

En caso de querer inscribirse a uno de los cursos se debe usar el comando:
node principal.js inscribir -c=3 -d=123 -n=andres

donde se pueden usar los alias o los nombres completos de los atributos que son:
-c, --curso
-n, --nombre,
-d, --documento

Todos son obligatorios.
